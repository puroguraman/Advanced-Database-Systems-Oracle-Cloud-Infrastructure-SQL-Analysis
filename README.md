📊 Oracle Cloud Database Management & SQL Analytics

📝 Project Overview:
This project focused on the advanced design, implementation, and management of a relational database system hosted on Oracle Autonomous Database (OCI). Developed as part of the "Advanced Database Systems" course (PSS-183) at the University of Piraeus, it covers the entire lifecycle of a database, from schema creation to complex data analysis and security configuration

🚀 Key Technical Features
Oracle Cloud Infrastructure (OCI): Deployment and management of a cloud-native autonomous database
Secure Connectivity: Implementation of secure communication using Oracle Wallets (Mutual TLS) and JDBC properties for enterprise-grade security
Advanced SQL Programming: Development of complex scripts featuring: Triggers for automated data integrity
Stored Procedures and Functions for backend logic
Sophisticated JOINs and Analytical Queries for business intelligence. Data Integrity & Backup: Comprehensive database backup strategy and schema versioning

Oracle Autonomous Database (19c/21c) 
Language: SQL, PL/SQL
Tools: Oracle SQL Developer, OCI Console
Security: mTLS (Wallet-based authentication), Encryption at rest
